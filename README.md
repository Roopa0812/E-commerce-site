# 🛒 E-Commerce Website

A multi-page **E-Commerce Web Application** built using **HTML, CSS, JavaScript, and Node.js**, designed to simulate a real-world online shopping experience.

This project demonstrates core e-commerce functionalities such as product browsing, cart management, user authentication pages, and wishlist handling.

---

## 🌐 Live Demo

🔗  https://e-commerce-project-812.netlify.app/ 

---

## 📌 Project Overview

This project is a frontend-focused e-commerce platform with basic backend support. It allows users to explore products, manage their shopping cart, and interact with different sections of an online store.

It is built to showcase:

* Frontend development skills
* UI/UX structuring
* JavaScript-based interactivity
* Basic backend and database integration

---

## ✨ Features

* 🛍️ Browse products
* 🛒 Add/remove items from cart
* ❤️ Wishlist functionality
* 🔐 User login and registration pages
* 🔄 Multi-page navigation
* ⚡ Dynamic UI updates using JavaScript
* 📊 Data integration using Excel & SQL

---

## 🧱 Tech Stack

* **Frontend:** HTML5, CSS3, JavaScript
* **Backend:** Node.js (Basic server setup)
* **Database:** MySQL (Database.sql)
* **Data Source:** Excel file (Ecommerce.xlsx)

## ⚙️ How to Run the Project

### 🔹 Run Frontend Only

1. Clone the repository:
   git clone https://github.com/Roopa0812/E-commerce-site.git

2. Open index.html in your browser

---

### 🔹 Run with Backend (Node.js)

1. Install Node.js

2. Clone the repository:
   git clone https://github.com/Roopa0812/E-commerce-site.git

3. Navigate to the project folder:
   cd E-commerce-site

4. Install dependencies:
   npm install

5. Start the server:
   node app.js

---

## 🗄️ Database Setup

1. Open MySQL or any SQL tool
2. Import Database.sql
3. Configure your database connection inside app.js
